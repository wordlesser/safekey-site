Frequently Asked Questions

I forgot my master password. Can I recover it?

No. SafeKey is designed so that your master password is never stored anywhere — not even by us. If you forget it, the encrypted data cannot be recovered. We recommend writing it down and storing it somewhere safe offline.

Where is my data stored?

Everything is stored locally on your device using AES-256-GCM encryption. Nothing is sent to any server or cloud service.

How do I back up my passwords?

Go to Settings > Data Management > Export. SafeKey will generate an encrypted backup file that you can store anywhere you like (Files app, iCloud Drive, etc.).

Face ID / Touch ID is not working.

Make sure you have enabled biometrics in iOS Settings and that you have turned on the Face ID toggle inside SafeKey's Settings. On first enable, you will be asked to verify with Face ID once to confirm.

The app locked me out after too many wrong attempts.

After 5 consecutive wrong password attempts, SafeKey temporarily locks for 30 minutes as a security measure. Please wait for the timer to expire and try again.

How do I change my master password?

Go to Settings > Security > Change Master Password. You will need to enter your current password first. All stored passwords will be re-encrypted automatically.

Still need help?

If your question is not answered above, please open an issue on GitHub and we will get back to you as soon as possible.
